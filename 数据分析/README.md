### 1. Numpy的安装
`windows+R`键调出运行框，输入`cmd`回车，然后在命令行输入：
```bash
pip3 install numpy -i https://mirrors.aliyun.com/pypi/simple/
```

回车进入安装，我之前已经安装完成了，所以提示`numpy`已经存在。

检测是否安装完成：
在命令行输入`python3`回车，在输入`import numpy` 回车，不报错说明已经大功告成。
